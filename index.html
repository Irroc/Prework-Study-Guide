HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • H1 though h6 adjust size of font from largest to smallest respectively.
  • Meta is information stored in the head.
  • Title is what appears on the tab of the webpage, and is stored in the head.
  • Ul is what starts a list.
  • Li is list items in a list.
  • Link creates a connection to an external file.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can refer to a color by its hexadecimal code to get the color you want.
  • Css rul is "selector(s) {property:value; property: value;}.
  • A declaration is the {property(s):value(s);} in the css rule.
  • It is important to have dry code and not unnecessarily repeat code separately that can otherwise be grouped together.
  • It is easier to organize your css code by linking a seperate file than to try to use it in the html file.

Git

  • The command "git status" checks what branch im currently in.
  • The command "git checkout -b 'insert branch name'" creates and moves my to the new branch.
  • The command "code ." pulls up my workstation in visual studio code.
  • The command "git checkout main" brings me to the main branch.
  • The command "git add -A" stages all saved files. git WILL NOT save changes if its not staged before committing.
  • The command "git commit -m 'insert discription of changes'" will commit changes to git as long as staged properly.
  • The command "git pull origin main" will pull the lates version of git to verify your up to date.
  • The command "git push origin 'current branch name'" will push it to git hub
  • The command "touch 'insert file name'" makes a new file.

JavaScript

  • JavaScript reads and executes from top to bottom, which is called control flow.
  • In "var shape = 'circle'" 'var' makes a new variable, shape is the name of the variable, and circle is data of the variable.
  • If statements interrupt control flow and create a loop in which conditionmust be met giving a form of logic to programs.
  • If statemets can have multiple conditions using 'else if' statements. note: 'else if' statements are also subject to control flow.
  • An array is a single variable that holds a group of data instead of one singular piece of data. IE: var shapes = ["triangle", "circle", "square"]
  • In an array theres index numbers automatically assigned to each piece of data starting at 0 and counting up by 1 following control flow.
  • You can list the data bits in arrays by using the 'for' stastement and the following three steps: Starting point of the loop, the condition of the loop, and the final step allows the araray to iterate over each item. IE: for(var x = 0; x < shapes.length; x++) { console.log(shapes[x]);
